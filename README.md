# Выбор метрик
Задача классификации <br/>
| Метрика | Описание |
| --- | --- |
| Accuracy | Доля верных предсказаний от общего числа предсказаний|
| F1 | Гармоническое среднее между точностью и полнотой модели |

| Метрика | Описание |
| --- | --- |
